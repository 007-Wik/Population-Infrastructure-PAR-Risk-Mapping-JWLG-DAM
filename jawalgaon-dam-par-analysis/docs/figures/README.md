## Recommended Export Settings

When exporting visualisations from the notebook for publication:
- Resolution: ≥ 300 DPI for print, 150 DPI for digital EAP
- Format: PNG (maps), SVG (charts/diagrams)
- Coordinate grid: Include UTM grid with 5 km intervals
- Scale bar and north arrow: Required for all maps
- Legend: Include all hazard classes, even those absent from the study area
