# Data Directory

## Structure

```
data/
├── raw/                            # Source data (inputs to the notebook)
│   └── Jawalgaon_PAR_Complete.zip  # All shapefiles and hydraulic raster outputs
│
└── outputs/                        # Generated by the notebook (not tracked in Git)
    ├── shapefiles/                 # Vector outputs: inundation extents, settlements
    ├── csv/                        # Tabular outputs: PAR tables, Annexures, summaries
    └── rasters/                    # Raster outputs: hazard classification, arrival time
```

## Raw Data Contents (`Jawalgaon_PAR_Complete.zip`)

### Shapefiles
| File | Description |
|------|-------------|
| `settlements_piping.shp` | Settlements with PAR attributes — piping scenario |
| `settlements_overtopping.shp` | Settlements with PAR attributes — overtopping scenario |
| `settlements_smoothed.shp` | Chaikin-smoothed settlement polygons (pre-analysis) |
| `inundation_piping.shp` | Flood extent — piping scenario |
| `inundation_overtopping.shp` | Flood extent — overtopping scenario |
| `roads_inundation.shp` | Road segments with flood attributes |
| `osm_poi_inundation.shp` | OSM POI with flood attributes |
| `dam_location.shp` | Dam centroid point |

### Rasters
| File | Description |
|------|-------------|
| `hazard_piping.tif` | H1–H6 classified hazard raster — piping |
| `hazard_overtopping.tif` | H1–H6 classified hazard raster — overtopping |
| `arrival_piping_relative.tif` | Arrival time (hrs after breach) — piping |
| `arrival_overtopping_relative.tif` | Arrival time (hrs after breach) — overtopping |
| `_reproj/` | Intermediate reprojected rasters (UTM) |

### CSVs
| File | Description |
|------|-------------|
| `Annexure_3A_Piping.csv` | EAP Annexure 3A — piping |
| `Annexure_3A_Overtopping.csv` | EAP Annexure 3A — overtopping |
| `Annexure_3B_Piping_PAR.csv` | EAP Annexure 3B — piping |
| `Annexure_3B_Overtopping_PAR.csv` | EAP Annexure 3B — overtopping |
| `Zonal_Statistics.csv` | Depth/velocity/WSE/arrival per settlement |
| `Master_Distance_From_Dam.csv` | All features by river chainage |
| `Critical_Infrastructure_At_Risk.csv` | Schools, hospitals, wells at risk |
| `Roads_Inundation.csv` | Road inundation by segment |
| `Flood_Wave_Travel_Time.csv` | Travel time table |
| `Road_Type_Summary.csv` | Inundated road length by type |
| `OSM_Water_Wells.csv` | Water wells at contamination risk |
| `OSM_POI_Flood_Status.csv` | All OSM POI with flood status |
| `OSM_POI_Category_Summary.csv` | POI count by category |

## Outputs Directory
The `outputs/` directory is excluded from Git (see `.gitignore`). It is regenerated by running the notebook from start to finish. All final outputs are also available in `raw/Jawalgaon_PAR_Complete.zip`.
